# ButanolPathOptimizer 🧪

Автоматический пайплайн для оценки бесклеточного мультиферментного синтеза n-бутанола.

## Что делает

Запускает 4 последовательных модуля и генерирует PDF-отчёт:

1. **FBA** (COBRApy) — стехиометрическое моделирование, теоретический выход бутанола
2. **Термодинамика** — расчёт ΔG реакций, выявление узких мест пути
3. **ODE-кинетика** (SciPy) — моделирование за 24 ч, свободные vs иммобилизованные ферменты
4. **Отчёт** (ReportLab + Matplotlib) — итоговый PDF с графиками и таблицами

## Быстрый старт

### Вариант 1 — автоматический скрипт

**Windows** (PowerShell):
```powershell
.\run.ps1
```

**Linux / macOS** (Terminal):
```bash
bash run.sh
```

### Вариант 2 — вручную

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ВАШ_ЛОГИН/butanol-pipeline.git
cd butanol-pipeline

# 2. Собрать образ
docker build -t butanol-pipeline .

# 3. Запустить
docker run --rm -v "${PWD}/output:/app/output" butanol-pipeline
```

После запуска откройте папку `output` — там будет файл `report.pdf`.

## Структура проекта

```
butanol_pipeline/
├── data/
│   └── params.csv        # входные параметры ферментов
├── modules/
│   ├── fba.py            # модуль 1: FBA
│   ├── thermo.py         # модуль 2: термодинамика
│   ├── ode.py            # модуль 3: ODE-кинетика
│   └── report.py         # модуль 4: генератор отчёта
├── output/               # сюда сохраняется report.pdf
├── main.py               # запускает пайплайн
├── requirements.txt      # зависимости Python
├── Dockerfile
├── run.ps1               # скрипт запуска для Windows
└── run.sh                # скрипт запуска для Linux/macOS
```
